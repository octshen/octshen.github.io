一种数组多项排序的实现

Posted on 2017-09-11

JavaScript中数组排序 sort 方法的参数是一个函数，sort 的排序实现是根据该函数的返回值进行的。
若 a 小于 b，在排序后的数组中 a 应该出现在 b 之前，则返回一个小于 0 的值。
若 a 等于 b，则返回 0。
若 a 大于 b，则返回一个大于 0 的值。
如果数组中包含字符串则按照每项首字母在ASCII码表中的大小进行排序

自定义的排序方法 arraySort

arraySort(arr, propOrders)

arr —- 要排序的数组
propOrders —- 排序条件
